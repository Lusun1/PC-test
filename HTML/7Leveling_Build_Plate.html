Leveling the Building Platform

After mounting the recoater blade, the next preparation step is leveling the building plate. The build will tilt or fail if the building plate is not flat. If the height of building plate is not correct, the recoater on the machine will crash into it. In this chapter, you will learn how to level the building platform.

Step1: Select the build plate on the left and place it on the building platform highlighted below.

Step2: Install the bolts to mount the build plate to the building platform. This fixes the build plate on the building platform.

Step3: Lower the build plate using the 'move platform down' button on the monitor to the right. This ensures that the recoater will not impact the build plate when moving over it.

Step4: Use the 'run recoater left' button on the monitor to move the recoater blade so it is over the build plate. This allows the user to place the dial indicator on the recoater.

Step5: Select the dial indicator to the left and attach it to the recoater housing. The magnet under the indicator will allow it to stay on the recoater housing.

Step6: Move the dial indicator back and forth using the computer mouse along the y-axis. Pay attention to the value on the indicator. If the value changes, you need to adjust the building platform until it is flat.

Step7: Adjust the building platform using the rocker switches on the bottom until the dial indicator's value is stable while moving back and forth.

Step8: In the real operation, you need to close the Process Chamber door in order to move the recoater from left to the right. But in our simulation, you could press the button on the screen to move the recoater left or right.

Step9: Adjust Level Rocker Switches until the Indicator's value is stable while moving.

Step10: Find 0.55mm Feeler Gauges to check the distance between Recoater and Buld Plate.

Step11: remove indicator.
