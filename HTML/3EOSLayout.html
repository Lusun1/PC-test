Touch sensitive monitor Maybe use Unit Model to represent the model
This touch sensitive screen is on the right side of the machine, which could be used to control the machine and establish the build. The following operations will be useful in your using of monitor:

  • Establish process conditions Start button
  • Pick the job
  • Stop when you have problem
  • Restart the build after you stop it


Emergency stop circuit Maybe use Unit Model to represent the model

By pressing the EMERGENCY STOP button on the machine all axis movements are stopped, the laser shut down and the process chamber door unlocked. Building platform heating, compressed air supply and Cooling system are also shut down. The operation of the Recirculating filter system is also stopped. If the emergency stop circuit is triggered an error message is displayed on the monitor.


Process chamber

Building process and operations happen inside the chamber.


Inert gas selection switch

The inert gas monitoring ensures the appropriate inert gas for the material selected in the building task has been selected. If the wrong inert gas has been selected, it is not possible to start the building process.
