Safety

The machineâ€™s safety concept is responsible for the protection the operator and protects him/her as well as possible.

There are several safety concepts you should know before using EOS

  • Chemical hazard
  • Fire fighting
  • Safety operation items:
    Emergency stop circuit
    Inert gas monitoring
    Low Oxygen alarms
    Wet separator
    Class D fire extinguisher

Chemical Hazard

  • Metal Powder and metal condensate
    Skin and eye contact with metal powder and metal condensate, as well as inhaling and swallowing metal powder can harm the health
    The powder inhalation hazard is the greatest when loading powder into the machine or manipulating powder. The associated procedure require the use of respirator.
  • Two classes of powders are used: Class A and Class B:
    Class A powders are steel and stainless steel
    Class B powders are aluminum, Titanium, Bronzer.
  • When we run the class A powder, we need to stay the class A powder. Because there is no need to change the filter . But when we change the filter from Class A to class B, the filter need to be changed. Class B powder is more dangerous than Class A powders. If you did not change the filter, it is more likely to cause fire because Class A and Class B cannot combine together.

Fire Fighting

  • Wire suitable protective equipment
  • In case of a fire in the machine, closing the process chamber door can help to prevent the fire spreading
  • For metal fires only use a suitable special fire extinguisher of fire class D.
  • Do not extinguish using water. Due to the reactivity of the metal powder with the water there is a risk of explosion.
  • Do not clean up burning metal powder using the Web separator, since using the Wet separator to clean up the burning metal powder can result in a deflagration.

Process Chamber Issue:
After starting the building process, the process chamber door is locked for safety reasons and cannot be opened.
Process chamber oxygen safety circuit: the oxygen safety circuit is a safety measure again a metal powder and metal condensate fire as well as the formation of a potentially explosive atmosphere by swirled up metal powder in combination with air. The oxygen safety circuit is triggered as soon as the oxygen content in the process chamber exceeds the threshold of 2.5 percent

Low Oxygen Alarm
The installation of low oxygen alarms in all rooms where argon could escape in the case of a leak from the supply pipe or the waste gas pipe.

Wet Seperator
The wet separator is used to cleaned up metal powder and metal condensates. But do not clean up burning metal powder using the Web separator.
